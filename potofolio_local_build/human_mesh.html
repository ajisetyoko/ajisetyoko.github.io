 3D Point Cloud Generative Adversarial Network from Human Action based Still Image 

Research Project Plan - Summer 2020 - SIMSLAB Laboratory
Aji Setyoko,  Hendrik Tampubolon 
National Taiwan University of Science and Technology, Taiwan

Project Description

Date: 1 June 2020 - Present

Goal:
1. Built a action recognition model based on 3d point cloud of still image data.
2. Built a Generative Adversial Network to generate an 3D point cloud data.

Progress:
The progress can be see in the CODE page. 
[30-June-2020] - Point Cloud action recognition dataset by PASCALL VOC 2007 extraction complete. 
[25-June-2020] - HMR tuning.
